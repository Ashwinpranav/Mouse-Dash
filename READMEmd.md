# 🐭 Mouse Dash

Mouse Dash is a simple and fun endless runner game built using HTML, CSS, and JavaScript.

## 🎮 Gameplay

- 🐭 Control a running mouse.
- 🏃 The mouse runs automatically.
- ⬆️ Press *Space* or *Tap* to jump.
- 🚧 Avoid obstacles.
- 🐱 If you hit an obstacle, the cat catches the mouse.
- 📈 Earn points by surviving longer.
- 🏆 Beat your high score.

## ✨ Features

- Endless runner gameplay
- Smooth jump mechanics
- Collision detection
- Score counter
- High score system
- Responsive design
- Mobile touch support
- Game Over and Restart

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript

## 🚀 Getting Started

1. Clone the repository
2. Open index.html in your browser
3. Start playing!
